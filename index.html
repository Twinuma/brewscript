brewscript

Description

By running this script, execution is possible with a single command with the following content. Please try it by all means everyone.

  • brew update
  • brew upgrade
  • brew doctor
  • awscli upgrade
  • wp-cli update
  • update_rubygems
  • gem update
  • grunt update
  • grunt-cli update
  • grunt-init update
  • bower update
  • less update
